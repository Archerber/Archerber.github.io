python之request库

Posted on 2021-04-09 | In web | Visitors: |
request

最基本的请求
get r=requests.get(url=’baidu.com’)
post请求 r = requests.post(‘https://www.csdn.net/', data = {‘key’:’value’})
put请求 r = requests.put(‘https://www.csdn.net/', data = {‘key’:’value’})
delete请求 r = requests.delete(‘https://www.csdn.net/')
head r = requests.head(‘https://www.csdn.net/')
option r = requests.options(‘https://www.csdn.net/')

print(r.text)打印解码后的返回数据

print(r.status_code)获取返回状态(如，404，200，详情见https://www.cnblogs.com/jinjiangongzuoshi/p/3778883.html)
print(r.encoding)返回编码方式
request的参数有params，用法:payload = {‘key1’: ‘value1’, ‘key2’: ‘value2’} r = requests.get(“http:///“, params=payload) print(r.url)
headers url = ‘https://api.github.com/some/endpoint' headers = {‘user-agent’: ‘my-app/0.0.1’} r = requests.get(url, headers=headers)
r.raw #返回原始响应体，也就是 urllib 的 response 对象，使用 r.raw.read() 读取
r.content #字节方式的响应体，会自动为你解码 gzip 和 deflate 压缩
r.headers #以字典对象存储服务器响应头，但是这个字典比较特殊，字典键不区分大小写，若键不存在则返回None

发送cookie
url = ‘http://httpbin.org/cookies'
cookies = dict(cookies_are=’working’)
r = requests.get(url, cookies=cookies)
r.text
‘{“cookies”: {“cookies_are”: “working”}}’
