另外一种思路，这个改下libc本地就可以打通了

off-by-null

只能溢出Null的情况,有两种利用思路

poison null byte

原理:堆布局如下
chunk a pre_size
size = 0x101
data
chunk b pre_size
size = 0x111
data
chunk c pre_size
size = 0x91
data
chunk d pre_size
size = 0x91
…
思路是free(b)，然后edit(a)把b的size置为0x100,然后malloc(0x80)(b1)、malloc(0x40)(b2)
接着free(b1)、free(c)，malloc(0x190)得到的堆就可以overlap了

house of einherjar

原理:堆布局如下
chunk a pre_size
size = 0x101
data
chunk b pre_size
size = 0x101
data
chunk c pre_size
size = 0x91
data
chunk d pre_size
size = 0x91
…
先free(a)、然后off-by-null把b的size设为0x100,再free(c),此时malloc(0x300)就可以控制b
例题祥云杯PassWordBox_FreeVersion(见祥云杯复现)
